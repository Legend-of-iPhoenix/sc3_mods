So basically I wrote a set of mods for SourceCoder3

and I decided to bundle them into a package. You're welcome.

Simply drag this link to your bookmarks bar to save it. While on the sc3 page, you can click the bookmark to launch the userscript.

You can check out the source on github.

I'll try to update it so that it works with future versions of sc3 if that's a thing, but I offer no guarentees.

No, I don't spy on your projects or anything, don't be ridiculous :P

None of my scripts cause any additional load on the server. Everything is done client-side.

List of mods:

  • Save project locally
  • Fullscreen SC3 View
  • Import / export project as text
  • Disable that annoying autosave shifting
  • Pick from 60 different CodeMirror themes
  • 1-click TI-BASIC code formatting
